Teoria

Wielomiany ortogonalne to rodzina wielomianw, w ktrych iloczyn skalarny dwch rznych wielomianw jest rwna 0.

Jeeli dodatkowo iloczyn skalarny wielomianu "przez siebie" jest rwny 1, to nazywamy je wielomianami ortonormalnymi

Iloczyn skalarny wielomianw mona zdefiniowa jako:

$<P_m,P_n>=\int_a^b P_n(x)P_m(x)W(x)dx = 0$

Funkcja W jest nazywana funkcj wagow.

Przykadowe rodziny wielomianw ortogonalnych

Nazwa Przedzia Funkcja wagowa Zastosowanie Parzysto
Czebyszewa (1 rzdu) [-1,1] $\frac{1}{\sqrt{1-x^2}}$ Zera wielomianu jako wzy interpolacji wielomianowej Dla parzystego k - parzyste, dla nieparzystego k - nieparzyste
Legendre [-1,1] 1 Zera wielomianu jako wzy kwadratur Gaussa-Legendre'a jw
Hermite [$-\infty$,$\infty$] $\exp(-x^2)$ Zera wielomianu jako wzy kwadratur Gaussa-Hermite'a jw
Laguerre [0,$\infty$] $\exp(-x)$ Zera wielomianu jako wzy kwadratur Gaussa-Laguerra'a brak

Wielomiany Hermite'a

Zdefiniowane s rekurencyjnie jako:

$H_0(x) = 1$

$H_1(x) = 2x$

$H_{n+1}(x) = 2xH_n(x)-2nH_{n-1}$

  • 0bliczamy funkcj bazow uwzgldniajc czynnik wagowy. $<P_m,P_n>=\int_{-\infty}^{\infty} H_n(x)H_m(x)\exp(-x^2)dx = \int_{-\infty}^{\infty}\underbrace{H_n(x)\exp(-\frac{x^2}{2})}_{\text{funkcja bazowa}_n}\underbrace{H_m(x)\exp(-\frac{x^2}{2})}_{\text{funkcja bazowa}_m}$

  • Wyliczamy jej wartoci dla podanych punktw x

  • Otrzymany wektor wartoci normalizujemy

In[12]:
# obliczamy funkcj bazow
# argument 'norm'  na potrzeby wykresw
function base_fun(k, x, norm=true)
    H_k = basis(Hermite, k).(x)
    app = H_k .* exp.(-(x.^2)/2)
    # normalize to have norm 1
    if norm
        return normalize(app)
    end
    return app
end

OK, baza jest ortonormalna

Obliczamy kolejne wspczynniki aproksymacji. Dziki ortonormalnej bazie, poszczeglne wspczynniki s iloczynami skalarnymi wektora wartoci funkcji aproksymowanej oraz wektora bazowego (analogia: rzut wektora wartoci funkcji aproksymowanej w kierunku wektora bazowego).

Wektor bazowy jest znormalizowany $|\vec{j}|=1$, co upraszcza obliczenia.

Analogia w 2D

$\text{rzut }\vec{a} \text{ w kierunku } \vec{j} =|\vec{a}|\cdot cos(\angle(\vec{a},\vec{j}))=|\vec{a}|\cdot|\vec{j}|\cdot cos(\angle(\vec{a},\vec{j}))=dot(\vec{a},\vec{j})$
